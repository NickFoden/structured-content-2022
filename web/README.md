The website of [Structured Content 2022][sc2022] conference.

## Getting Started

Install dependencies:

```bash
npm i
# or
yarn
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
